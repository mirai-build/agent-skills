> 使い方: 非同期イベントや Webhook がある bounded context で追加する。送信 / 受信の契約を一覧化し、各契約の詳細文書への入口にする。

## 0. 文書の目的

この文書は、この bounded context に関係する非同期イベントや Webhook 契約を一覧化し、payload、配送保証、認証、障害時の扱いを追えるようにするための正本である。
